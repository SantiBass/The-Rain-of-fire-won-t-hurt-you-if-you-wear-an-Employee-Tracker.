## Installation
#### To run this application first run "[npm i node](https://www.npmjs.com/package/node)", "[npm i inquirer](https://www.npmjs.com/package/inquirer)" and  "[npm i mysql](https://www.npmjs.com/package/mysql)" in [Visual Studio Code](https://code.visualstudio.com/download) to install all dependencies and CLI capabilities. [Visual Studio Code](https://code.visualstudio.com/download) is a source-code editor made by Microsoft for Windows, Linux and macOS.

## Usage
#### In this simple application, the enter, up, and down arrow keys are used for navigation. Some data entry is needed to update, add, or delete information in the tables. For example, fields such as first name, manager ID, and role ID can be used to update, add, or delete information in the tables. To run the application, type in and press enter after these command lines: SOURCE schema.sql, SOURCE seeds.sql, SOURCE query.sql, and node index.js.
